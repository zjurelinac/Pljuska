Language

Overview and documentation of a HASH scripting language

Introduction

Hash is a simple and easy-to-use scripting language. It's syntax and commands are quite similar to other UNIX scripting languages, so if you've ever used any of them, say Bash or Zsh, you'll find Hash familiar and understandable.

The documentation is divided in two parts. The first deals with basic language syntax and features, while the second contains a short description and usage examples of every command supported by Hash.

Language overview

Here is a short overview of Hash language characteristics and features.

Syntax

Hash language consists of a series of statements, which can be either commands, assignments or control statements. Statements are most often separated by newlines, though semicolons ; can serve the same purpose.

Variables

Hash supports two separate data types: integers and strings, whereby string values must be enclosed into double quotes, "like this". Integer values support all the basic arithmetic operations: addition, subtraction, multiplication, division and modulo, while strings can contain variables whose values are inserted into it when the string is first used.

Variables in Hash can be of either string or integer type, but it's type is changeable during program execution. Variables are always identified by preceding a $ symbol in front of their name. Variable names can contain both uppercase and lowercase letters, digits and an underscore symbol _.

Beware though, some variable names are reserved for special purposes, like $0 which contains the script name, and $1, $2..., where script arguments are automatically stored.

Arithmetic

Integer data type in Hash supports all basic arithmetic operations: addition +, subtraction –, multiplication*, division / and modulo %. There is also an unary - operator, which should be enclosed in parentheses together with the term it applies on, because otherwise there is a possibility of parse errors.

Several examples of arithmetic expressions:

$root = ( $a + $n / $a ) / 2
$year = ( $days - $days / 1460 + $days / 36524 - $days / 146097 ) / 365
