Introducing Hash and Pljuska

A simple, fast scripting language with a familiar, BASH-like syntax, only further improved.

Runs on a lightweight interpreter written entirely in Haskell, and is available on almost every platform.

The interpreter can be downloaded for free from the Github repo.

Both the language and the interpreter were made as a project for a
Programming in Haskell course on FER Zagreb.
